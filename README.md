# Resume Scorer Frontend

A modern React frontend built with Tailwind CSS for the Resume Scorer application.

## Features

- **Modern UI**: Clean, responsive design with Tailwind CSS
- **Resume Upload**: Drag-and-drop file upload interface
- **Job Search**: Multi-platform job search with real-time results
- **ATS Scoring**: Visual score displays with progress bars
- **Analytics Dashboard**: Comprehensive analytics and insights
- **Mobile Responsive**: Works seamlessly on all devices

## Tech Stack

- React 18
- Tailwind CSS 3.4
- React Router DOM
- Axios for API calls
- Lucide React for icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Components

- **Home**: Landing page with features overview
- **Upload**: Resume upload interface
- **Results**: Job search and ATS scoring
- **Analytics**: Performance analytics and insights

## API Integration

The frontend communicates with the Django backend through RESTful APIs:

- Resume upload and analysis
- Job search across multiple platforms
- ATS score calculation
- Analytics and insights

## Styling

Uses Tailwind CSS utility classes for:
- Responsive design
- Component styling
- Color schemes
- Animations and transitions
